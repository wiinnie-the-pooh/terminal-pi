# Changelog

All notable changes to Pi Coding Agent CLI are documented here.

## [0.2.4]

### Changed
- README: use PNG icon instead of SVG theme icons for marketplace compatibility

## [0.2.3]

### Changed
- README: rewrite with workflow-focused user guide

## [0.2.2]

### Added
- Pi terminals now export `EDITOR` and `VISUAL` so Pi's external-editor flow opens in VS Code
- New `piCodingAgent.editorCommand` setting to configure the editor command

## [0.2.1]

### Added
- `Ctrl+G` keybinding sends Pi's external-editor control sequence when a Pi terminal is focused
- Pi icon in editor title bar for one-click launch
- Each Pi command always opens a fresh terminal
